# Venkatesh Chenna Portfolio Website

A professional, responsive portfolio website for Venkatesh Chenna, Full Stack .NET Developer.

## Files

- `index.html` - Main portfolio page
- `styles.css` - Website styling
- `script.js` - Mobile menu and scroll animations
- `assets/venkatesh-hero.jpg` - Portfolio profile image
- `assets/Venkatesh_Chenna_Resume.pdf` - Resume for download

## GitHub Pages Hosting Steps

1. Create a new GitHub repository.
2. Upload all files from this folder to the repository root.
3. Go to repository **Settings**.
4. Open **Pages**.
5. Under **Build and deployment**, choose:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
6. Save.
7. Wait 1–3 minutes. GitHub will show the live website URL.

## Important

Keep `index.html` in the root folder. Do not place it inside another folder, otherwise GitHub Pages may not open the site correctly.
